# Installation

- fork the repo
- clone your repo ```git clone https://github.com/<username>/UnitySpace.git``` on your terminal
- ```cd UnitySpace```
- ```cd client```
- ```npm install``` to download all dependencies
- ```npm run dev``` to start the development build.
